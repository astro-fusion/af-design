MCP Server

Model Context Protocol server for AI agents to query design tokens. Works with Cursor, Claude Desktop, VS Code, Windsurf, and more.

Installation

npx @astrofusion/design-mcp-server start

Configuration (mcp.json)

{
"astrofusion-design": {
"command": "npx",
"args": ["@astrofusion/design-mcp-server", "start"]
}
}

Available Tools

get_component_prompt Get AI prompt for a specific component on a platform
get_platform_context Get full system prompt for building UI on a platform
