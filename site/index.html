AstroFusion Design System

LLM-First Design System. Built for AI agents. Generative UI ready. MCP enabled.

A design system built with AI and LLMs in mind ‚Äî enabling agents to query tokens via MCP, generate safe UI with A2UI, and maintain design consistency across human and machine collaborators.

üì± Cross-Platform (Encore Pattern)

Same API, different renderers. Web uses Tailwind, Native uses NativeWind. One design system, two platforms.

.tsx React (Web) 
// Button.tsx (Web)
import { cn } from "./utils";

export const AFButton = ({ variant, children }) => (
  <button className={cn(
    "rounded-lg px-4 py-2 font-medium",
    variant === "default" && "bg-primary text-white",
    variant === "outline" && "border border-primary"
  )}>
    {children}
  </button>
);
.native.tsx React Native 
// Button.native.tsx (Mobile)
import { Pressable, Text } from "react-native";

export const AFButton = ({ variant, children }) => (
  <Pressable className={cn(
    "rounded-lg px-4 py-2",
    variant === "default" && "bg-primary",
    variant === "outline" && "border border-primary"
  )}>
    <Text className="font-medium">{children}</Text>
  </Pressable>
);

MCP Server

                
Model Context Protocol server for AI agents to query design tokens. Works with
                    Cursor, Claude Desktop, VS Code, Windsurf, and more.

            


            

                
Installation

                
npx @astrofusion/design-mcp-server start

            


            

                
Configuration (mcp.json)

                
{
  "astrofusion-design": {
    "command": "npx",
    "args": ["@astrofusion/design-mcp-server", "start"]
  }
}
